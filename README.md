# Tarea Semana 6

Este proyecto es una aplicación en C# Windows Forms que permite registrar y gestionar alumnos y profesores.  
El sistema utiliza herencia y polimorfismo para diferenciar los tipos de registros.  

## Características
- Registro de alumnos (con nombre, edad, carrera y materia).
- Registro de profesores (con nombre, edad, especialidad y materia).
- Visualización en un `DataGridView`.
- Uso de `BindingList` para enlazar datos dinámicamente.
- Botones para agregar, actualizar limpiar y eliminar registro.
- Funcionalidad para seleccionar y mostrar detalles en formularios.
